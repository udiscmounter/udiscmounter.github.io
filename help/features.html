Key Features

Manage Virtual Drives

Set up uDiscMounter virtual drives on Windows starting from installing virtual drive drivers and uninstalling the drivers. and set the number of virtual drives according to user needs.

Mount and Eject Disk image files

Mount a disk image file to a virtual drive to virtualize the disk image file, and to eject the disk image file on the virtual drive.

Associate files with uDiscMounter

uDiscMounter can be associated with almost all disk image formats.
