HOW TO ADD VIRTUAL DRIVE IN UDISCMOUNTER

Adding a virtual drive in uDiscMounter is very simple. First, run uDiscMounter with administrator privileges by right-clicking the application and selecting "Run as Administrator." Then, go to the "Add Virtual Drive" menu and specify the number of virtual drives you want to create, as shown in the image below.

Note: After making changes to the virtual drives, please restart your device to ensure that all virtual drives function properly.
